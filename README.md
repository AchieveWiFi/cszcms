CSZ-CMS


Content Management System (CMS) with Codeigniter and Bootstrap. This is Open-Source for free.

Server Recommend required:
- PHP 5.3 or higher.
- MySQL 5.0 or higher (Recommend for MySQLi Driver).
- PHP Shorttag Enable.
- PHP GD Enable.
